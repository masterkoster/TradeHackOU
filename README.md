# TradeHackOU

Minimal hackathon trading UI with local auth and Alpaca market data (planned).

## Features
- Email/password auth (local only)
- Login and registration start page
- Protected dashboard placeholder

## Setup
1) Install dependencies

```bash
npm install
```

2) Run the app

```bash
npm run dev
```

## Notes
- Auth data lives in localStorage for hackathon use only.
- Charts and realtime data will be added next.
